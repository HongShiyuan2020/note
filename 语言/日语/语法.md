## 用「だ」表示某物的状态

1. 用「だ」表示某物的状态：把「だ」加到名词或者な形容词后面  
    例：人＋だ＝人だ
2. 表示否定状态的活用规则：将「じゃない」加在名词或者な形容词后面  
    例：学生＋じゃない＝学生じゃない
3. **状态的过去形：** 把「だった」加到名词或者な形容词后面。
    例：友達＋だった ＝友達だった
4. 状态的否定过去形：先把名词或者な形容词变成否定式，然后把「じゃない」的「い」换成「かった」  
    例：友達じゃない → 友達じゃなかった＝友達じゃなかった

> **表示状态的时候可以把「だ」都省去！**

|     | 肯定    |       | 否定       |        |
| --- | ----- | ----- | -------- | ------ |
| 非过去 | 学生（だ） | 是学生   | 学生じゃない   | 不是学生   |
| 过去  | 学生だった | 以前是学生 | 学生じゃなかった | 以前不是学生 |

## い形容词

> 与な形容词不同，你**无需**在用い形容词修饰名词的时候添加一个「な」。
> 所有い形容词都是以平假名「い」结尾的。
> ****不要在い形容词后面加「だ」！****

1. **否定式**：首先把结尾的「い」去掉，然后加上「くない」。
2. **过去形**：首先把い形容词或否定い形容词结尾的「い」去掉，然后加上「かった」

|      | 肯定   | 否定     |
| ---- | ---- | ------ |
| 非过去形 | 高い   | 高くない   |
| 过去形  | 高かった | 高くなかった |

- 例外：
	- 良い　いい→よくない→よかった→よくなかった
	- 格好いい　かっこいい→かっこよくない→かっこよかった→かっこよくなかった

## る动词、う动词例外

る动词：所有る动词以「る」结尾
う动词：
1. 一个**不是**以「る」结尾的动词肯定是う动词。
2. 对于以「る」结尾的动词，如果在「る」音之前的假名包含 /a/、/u/ 或者 /o/ 元音，那它就是う动词，而前面是 /i/ 或者 /e/ 元音的，那它**大多数情况下**就是る动词。
例外：する、来る

## 动词的否定式

> 例外：ある、いる
> 说这个插曲的原因是因为「ある」的否定式是「ない」（意思是某物不存在）。

1. る动词：把「る」去掉换成「ない」
2. う动词：
	1. **对以「う」结尾的う动词：** 把「う」换成「わ」，再加上「ない」

## 用「～たりする」表达多个动作或状态 （从句后）

> 这是助词「や」的动词版本。想从一堆动词里面挑几个连起来的话，只要挨个**活用成过去形**，然后**加上「り」**。在最后还要**加上动词「する」**。跟助词「や」一样，时态取决于最后的动词，也就是「する」（因为它必须加在末尾）。

2. 除了正在进行时外，还可以你完成了动作，然后处于那个状态

## 将动作动词（行く、来る）与て形合用

> 把动作动词「去」（行く）和「来」（来る）与て形合用，这时候就表示这个动作朝向或来自某个地方。
> 
> 动作动词也可以用在时间表达式里，表示未来的趋势，或者迄今为止的情况

## て形的功能

1. 连续动作或状态
2. 表示正在进行、或进行后处于的状态
3. 完成了某个状态
4. 为未来做准备
5. 结合「行く」「来る」表示动作的方向

## 可能形：表达做某事的能力

1. る动词：る→られる/れる
2. う动词：う段→え段＋る
3. 例外：
	1. する→できる
	2. くる→こられる

> 可能形可视为る动词
> 可能形表达的是可能性，不能用直接对象助词「を」

> 如果想说某物可能存在，「あり得る」「ありうる」「ありえる」。_但是_其活用形例如「ありえない」、「ありえた」和「ありえなかった」都只有一种读法，中间是「え」。

## 表示条件的助词(と、なら、ば形、たら形、もし)

1. 条件+と+结果，条件+だと+结果
2. 前提条件＋なら/ならば＋结果
3. ば活用：
	1. 动词：う段→え段＋ば
	2. い形容词和ない结尾：い→ければ
	3. 名词和な形容词：+であれば
4. たら活用：名词、形容词、动词活用为过去形＋ら

## 表达不能、必须、不得不、可以做、可以不做（だめ、いけない、ならない、、）

1. 不能做什么：动词て形+は+だめ/いけない/ならない
2. 必须、不得不做什么：
	1. 否定て形+は+だめ/いけない/ならない
	2. 动词否定+表条件と+だめ/いけない/ならない
	3. 动词否定+表条件ば+だめ/いけない/ならない
3. 简写：
	1. なくて→なくちゃ
	2. なけらば→なきゃ
	3. ては→ちゃ＋だめ（不能做什么）
	4. では→じゃ＋だめ（不能做什么）
4. 可以做、可以不做：
	1. て形＋も+いい/大丈夫/構わない
	2. 简写：
		1. てもいい→ていい
