## 布鲁姆认知目标分类（认知层次）

| 层次  | 英文原词           | 中文释义 | 典型行为动词        | 核心问题类型               |
| --- | -------------- | ---- | ------------- | -------------------- |
| 1   | **Remember**   | 记忆   | 识记、列举、定义、回忆   | “什么是……？”、“请写出……”     |
| 2   | **Understand** | 理解   | 解释、描述、归纳、举例   | “为什么……？”、“说明原理”      |
| 3   | **Apply**      | 应用   | 计算、推导、操作、使用   | “求……”、“计算……”         |
| 4   | **Analyze**    | 分析   | 比较、分类、探究、因果分析 | “比较两种情况”、“分析实验误差”    |
| 5   | **Evaluate**   | 评价   | 判断、评估、论证、选择   | “哪个更合理？”、“说明优劣并给出理由” |
| 6   | **Create**     | 创造   | 设计、改进、创新、综合   | “设计实验”、“提出改进方案”      |
## 具体类别
### T01. 现象解释与原理解释
理辨识类
**认知层级**：理解（Understand）→ 应用（Apply）
**核心考查**：能否用准确的物理概念和规律解释自然或实验现象，识别其背后的物理原理。  
**本质**：连接抽象知识与具体世界。  
**子类：**
1. **生活现象解释型**——从生活场景中提炼规律
2. **实验与器件原理识别型**——识别装置或现象的物理本质  
![[截屏2025-10-28 15.26.06.png]]
#### 例题
1. **（北京2024-8）“省油灯”原理**：解释向灯盏夹层注水为何能减慢油的蒸发。
2. **（重庆2024-2）砂锅煲沸腾**：解释停止加热后砂锅煲为何能继续沸腾。
3. **（安徽2018-2）硝化棉燃烧**：解释下压活塞时硝化棉燃烧的原因（做功改变内能）。
4. **（海南2018-16）游泳池边温差**：解释水泥地面烫脚而池水凉的原因（比热容）。

### T02. 概念辨析与逻辑判断类
**认知层级**：分析（Analyze）→ 评价（Evaluate）
**核心考查**：区分易混淆概念、判断规律适用范围、分析陈述的逻辑正确性。  
**本质**：考查对物理概念的精确理解与逻辑思维。
**子类：**
1. **概念辨析型**——区分易混概念或判断正确规律  
2. **正误判别**——发现并解释偏差  

#### 例题
1. **（北京2024-9）概念正误判断**：考查分子间作用力、热值、汽油机原理、吸热与温度关系等概念的辨析。
2. **（北京2024-10）滑板车中的物理**：判断关于做功、惯性、受力、运动状态改变的说法的正误。
3. **（安徽2021-11）观点判断**：判断关于气体流速与压强、惯性、电磁感应、安全用电的说法的正误。
![[截屏2025-10-28 15.26.45.png]]
### T03. 错误分析与故障判断类
**认知层级**：评价（Evaluate）
**核心考查**：基于异常现象或错误操作，识别错误、分析原因、提出改进方案。  
**本质**：逆向思维与系统诊断能力。（与概念辨析类的区别在于：**它要求从错误出发进行因果推理**，不是单纯判断对错。）
#### 例题
1. **（北京2024-11）电路故障分析**：根据电表示数变化，判断电路中电阻R1或R2的故障类型（短路或断路）。
2. **（安徽2021-15）电路故障判断**：根据电流表有示数、电压表无示数，判断R1或R2的故障。
3. **（广州2018-22(4)）电路保护**：分析滑动变阻器阻值为零时导致电源短路的原因。
![[截屏2025-10-28 15.29.28.png]]
### T04. 图像与数据理解类
**认知层级**：理解（Understand）→ 应用（Apply）
**核心考查**：从图像、数据或仪器读数中提取物理信息，并进行信息间的转换与表达。  
**本质**：由“视觉符号”到“物理意义”的转换能力。
**子类：**
1. **仪表与数据读数**  
2. **函数与图像理解**  
3. **图像信息转换与综合分析**  
#### 例题
1. **（北京2024-12）测温电路分析**：根据电阻R的阻值-温度图像和电压表示数变化，求解电流、电压并判断如何提高灵敏度。
2. **（重庆2024-13）电路图像分析**：利用小灯泡的I-U图像，结合电路安全条件，计算电源电压和电阻阻值。
3. **（安徽2021-20）I-1/R图像**：通过绘制和分析电流与电阻倒数的图像，探究电流与电阻的关系。
![[截屏2025-10-28 15.31.53.png]]
### T05. 实验探究与方法评价类
**认知层级**：分析（Analyze）→ 创造（Create）
**核心考查**：科学探究方法与实验思维，包括变量控制、方案设计、装置改进与结论推理。  
**本质**：体现科学探究全过程的合理性与创新性。
**子类**：
1. 实验控制与变量设计
2. 装置改进与方法优化
3. 结论推理与假设验证
4. 实验安全规范
#### 例题
1. **（北京2024-19）串联电路电流规律探究**：要求学生设计电路图和数据表格，探究串联电路中电流处处相等。
2. **（北京2024-23）密度计改进**：分析如何通过改变铁砂质量来增大密度计两刻度线间的距离。
3. **（福建2018-29）“密度计”制作**：利用弹簧测力计、木块等器材制作并标定“密度计”，考查装置改进与误差分析。
![[截屏2025-10-28 15.33.51.png]]

### T06. 定量计算与理论推证类
**认知层级**：应用（Apply）→ 分析（Analyze）→ 评价（Evaluate）  
**核心考查**：运用物理公式和数学推理解决数值或逻辑性问题。  
**本质**：模型化思维与演绎推理。
**子类**：
1. 基础计算类
2. 模型推导类
#### 例题
1. **（北京2024-25）电热器功率计算**：根据电路图计算低温档电流和高温档功率。
2. **（重庆2024-21）台灯电路计算**：计算正常发光时的电阻、产生的热量以及电压变化后的最小实际功率。
3. **（安徽2021-22）混联电路计算**：通过开关切换改变电路结构，利用串联电路规律计算电源电压和未知电阻。
![[截屏2025-10-28 15.34.18.png]]
### T07. 综合迁移与创新应用类
**认知层级**：评价（Evaluate）→ 创造（Create）  
**核心考查**：将学过的模型迁移到新情境中解决问题，体现跨知识整合与创新能力。  
**本质**：知识迁移与模型再造。
#### 例题
1. **（北京2024-26）东北虎质量估算**：创设“通过脚印估算老虎质量”的新情境，综合运用压强、重力等知识解决实际问题。
2. **（北京2024-15）彩球温度计**：将浮力、密度与温度测量创新结合，要求分析彩球状态与环境温度的关系，并提出改进最高测温范围的方法。
3. **（安徽2021-23）杠杆与悬吊木板模型**：将杠杆平衡模型迁移到悬吊的、有宽度的长方形木板上，通过理论推证判断其能否平衡，极具创新性。

## 认知层次与类别的对应关系

| 布鲁姆层次             | 主要考查能力         | 对应题型             |
| :---------------- | :------------- | :--------------- |
| **记忆 Remember**   | 识记概念、规律、符号     | （隐含于各题）          |
| **理解 Understand** | 解释、归纳、识别原理     | 现象解释与原理辨识类、图像理解类 |
| **应用 Apply**      | 运用规律、计算、数据处理   | 定量计算类、图像与数据理解类   |
| **分析 Analyze**    | 区分、推理、变量控制     | 概念辨析类、实验探究类      |
| **评价 Evaluate**   | 判断错误、故障诊断、方案评估 | 错误分析与故障判断类、探究类   |
| **创造 Create**     | 设计实验、迁移模型、综合创新 | 实验探究类、综合迁移类      |

| 类别         | 认知定位    | 主要考查能力     | 区分说明        |
| :--------- | :------ | :--------- | :---------- |
| 现象解释与原理辨识类 | 理解      | 从现象到规律的迁移  | 注重解释自然或实验现象 |
| 概念辨析与逻辑判断类 | 分析      | 辨析概念、逻辑判断  | 静态判断型思维     |
| 错误分析与故障判断类 | 评价      | 识别错误、逆向推理  | 从错误出发进行因果分析 |
| 图像与数据理解类   | 理解 / 应用 | 读图、转化、理解规律 | 符号信息→物理意义转换 |
| 实验探究与方法评价类 | 分析 / 创造 | 设计、改进、推理   | 体现科学探究全过程   |
| 定量计算与理论推证类 | 应用 / 分析 | 数学化推理、模型建构 | 定量与逻辑性最强    |
| 综合迁移与创新应用类 | 评价 / 创造 | 知识迁移与创新    | 跨学科融合与高阶能力  |
